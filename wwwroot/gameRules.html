Echo Words

Game rules 📏

Player one is asked to input a word and writes the first word, press Enter when finished and a timer will start (2-3 seconds) before the word disappear Player 2 first enters S1 word, then own word. First letter of S2 letter must match S1 last letter (and database of swedish dictionary) All players have iterated the array and entered new word. A countdown is started at every "input from user" 10 sec
